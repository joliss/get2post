get2post

A simple redirect service to convert GET requests to POST requests.

What is get2post?

get2post is an online utility that takes a URL with query parameters (normally used in GET requests) and redirects it to a POST request.

The main use case is using Chrome's site search functionality for websites that only support POST requests.

How to Use

Simply append your target URL after the hash (#) symbol of the URL https://joliss.github.io/get2post/redirect#, with any POST parameters included as GET parameters. For example:

https://joliss.github.io/get2post/redirect#https://httpbin.org/post?foo=x&bar=y

This will result in a POST request to https://httpbin.org/post, with the foo and bar parameters turned into POST data.

Privacy

get2post doesn't collect any data. Your requests and parameters are kept private because:

  • All data is processed entirely in your browser.
  • I don't log anythingâ€”analytics or otherwise.
  • GitHub's servers (where this tool is hosted) can see your HTTP request, but not the URL you're redirecting to. That's because the hash fragment (everything after the #) is not sent to the server.

Limitations

  • The target server must accept application/x-www-form-urlencoded POST data.
  • This tool doesn't support file uploads.
